# EnumFunctionStatus

High level indication of the final disposition of a backgrounded function - \"COMPLETED\": the request completed (not that the final result is a success, just that we completed) - \"FAILED\": the request failed.  - \"INCOMPLETE\": could not complete the request. 

## Enum

* `COMPLETED` (value: `'COMPLETED'`)

* `FAILED` (value: `'FAILED'`)

* `INCOMPLETE` (value: `'INCOMPLETE'`)
