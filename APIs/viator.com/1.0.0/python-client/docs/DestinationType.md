# DestinationType

**destination type specifier** - `\"COUNTRY\"`: destination is a country - `\"REGION\"`: destination is a recognized region of a country, such as a state, island within a country, stretch of coastline, etc. - `\"CITY\"`: destination is a city

* `CITY` (value: `'CITY'`)

* `COUNTRY` (value: `'COUNTRY'`)

* `REGION` (value: `'REGION'`)
