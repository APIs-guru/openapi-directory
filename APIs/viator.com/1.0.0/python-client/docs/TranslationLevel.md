# TranslationLevel

**numeric indicator** of the language translation level for *this* product that is one of: - `0`: no translation (English only) - `80`: full machine translation - `100`: full human translation  See: [Working with human and machine translations](#section/Appendices/Working-with-human-and-machine-translations) for more information 

## Enum

* `NUMBER_0` (value: `0`)

* `NUMBER_80` (value: `80`)

* `NUMBER_100` (value: `100`)
