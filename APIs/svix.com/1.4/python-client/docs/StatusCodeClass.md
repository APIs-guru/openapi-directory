# StatusCodeClass

The different classes of HTTP status codes: - CodeNone = 0 - Code1xx = 100 - Code2xx = 200 - Code3xx = 300 - Code4xx = 400 - Code5xx = 500

* `CodeNone` (value: `0`)

* `Code1xx` (value: `100`)

* `Code2xx` (value: `200`)

* `Code3xx` (value: `300`)

* `Code4xx` (value: `400`)

* `Code5xx` (value: `500`)
