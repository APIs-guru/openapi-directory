# ReplySettings

Shows who can reply a Tweet. Fields returned are everyone, mentioned_users, and following.

* `MENTIONEDUSERS` (value: `'mentionedUsers'`)

* `FOLLOWING` (value: `'following'`)

* `OTHER` (value: `'other'`)
