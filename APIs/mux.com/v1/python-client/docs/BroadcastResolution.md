# BroadcastResolution

The resolution of the composited video sent to the live stream. Defaults to `1920x1080` if not set.

## Enum

* `ENUM_1920X1080` (value: `'1920x1080'`)

* `ENUM_1280X720` (value: `'1280x720'`)

* `ENUM_1080X1920` (value: `'1080x1920'`)

* `ENUM_720X1280` (value: `'720x1280'`)

* `ENUM_1080X1080` (value: `'1080x1080'`)

* `ENUM_720X720` (value: `'720x720'`)
