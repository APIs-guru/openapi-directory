# LiveStreamStatus

`idle` indicates that there is no active broadcast. `active` indicates that there is an active broadcast and `disabled` status indicates that no future RTMP streams can be published.

## Enum

* `ACTIVE` (value: `'active'`)

* `IDLE` (value: `'idle'`)

* `DISABLED` (value: `'disabled'`)
