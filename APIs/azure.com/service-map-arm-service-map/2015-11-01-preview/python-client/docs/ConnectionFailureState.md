# ConnectionFailureState

Connection failure state:  * ```ok``` indicates no failures  * ```failed``` indicates only failures  * ```mixed``` indicates both failures and successes

## Enum

* `OK` (value: `'ok'`)

* `FAILED` (value: `'failed'`)

* `MIXED` (value: `'mixed'`)
