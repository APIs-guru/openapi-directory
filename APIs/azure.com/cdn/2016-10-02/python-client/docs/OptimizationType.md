# OptimizationType

Customer can specify what scenario they want this CDN endpoint to optimize, e.g. Download, Media services. With this information we can apply scenario driven optimization.

## Enum

* `GENERALWEBDELIVERY` (value: `'GeneralWebDelivery'`)

* `GENERALMEDIASTREAMING` (value: `'GeneralMediaStreaming'`)

* `VIDEOONDEMANDMEDIASTREAMING` (value: `'VideoOnDemandMediaStreaming'`)

* `LARGEFILEDOWNLOAD` (value: `'LargeFileDownload'`)

* `DYNAMICSITEACCELERATION` (value: `'DynamicSiteAcceleration'`)
