# OutputErrorPolicy

Indicates the policy to apply to events that arrive at the output and cannot be written to the external storage due to being malformed (missing column values, column values of wrong type or size).

## Enum

* `STOP` (value: `'Stop'`)

* `DROP` (value: `'Drop'`)
