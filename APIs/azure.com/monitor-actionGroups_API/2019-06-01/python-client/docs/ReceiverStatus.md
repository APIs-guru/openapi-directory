# ReceiverStatus

Indicates the status of the receiver. Receivers that are not Enabled will not receive any communications.

## Enum

* `NOTSPECIFIED` (value: `'NotSpecified'`)

* `ENABLED` (value: `'Enabled'`)

* `DISABLED` (value: `'Disabled'`)
