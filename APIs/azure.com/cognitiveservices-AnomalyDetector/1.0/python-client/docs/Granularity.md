# Granularity

Can only be one of yearly, monthly, weekly, daily, hourly or minutely. Granularity is used for verify whether input series is valid.

## Enum

* `YEARLY` (value: `'yearly'`)

* `MONTHLY` (value: `'monthly'`)

* `WEEKLY` (value: `'weekly'`)

* `DAILY` (value: `'daily'`)

* `HOURLY` (value: `'hourly'`)

* `MINUTELY` (value: `'minutely'`)
