# InstanceFlexibility

Allows reservation discount to be applied across skus within the same Autofit group. Not all skus support instance size flexibility.

## Enum

* `ON` (value: `'On'`)

* `OFF` (value: `'Off'`)

* `NOTSUPPORTED` (value: `'NotSupported'`)
