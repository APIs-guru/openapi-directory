# PaymentStatus

Describes whether the payment is completed, failed, cancelled or scheduled in the future.

## Enum

* `SUCCEEDED` (value: `'Succeeded'`)

* `FAILED` (value: `'Failed'`)

* `SCHEDULED` (value: `'Scheduled'`)

* `CANCELLED` (value: `'Cancelled'`)
