# CategoryPattern

The category of a recognition unit represents the type of content for that unit.

* `ROOT` (value: `'root'`)

* `WRITINGREGION` (value: `'writingRegion'`)

* `PARAGRAPH` (value: `'paragraph'`)

* `LINE` (value: `'line'`)

* `INKBULLET` (value: `'inkBullet'`)

* `INKDRAWING` (value: `'inkDrawing'`)

* `INKWORD` (value: `'inkWord'`)

* `UNKNOWN` (value: `'unknown'`)
