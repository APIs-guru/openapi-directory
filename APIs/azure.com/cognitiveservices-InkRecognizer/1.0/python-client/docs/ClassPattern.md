# ClassPattern

The class represents the type of the recognition unit. A recognition unit can be a leaf node or a container node. Container nodes typically have leaf nodes as children.

## Enum

* `CONTAINER` (value: `'container'`)

* `LEAF` (value: `'leaf'`)
