# PropertyBatchInfoKind

The kind of property batch info, determined by the results of a property batch. The following are the possible values.

## Enum

* `INVALID` (value: `'Invalid'`)

* `SUCCESSFUL` (value: `'Successful'`)

* `FAILED` (value: `'Failed'`)
