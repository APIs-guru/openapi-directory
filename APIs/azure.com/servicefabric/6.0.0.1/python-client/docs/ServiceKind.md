# ServiceKind

The kind of service (Stateless or Stateful). Following are the possible values.  - Invalid - Indicates the service kind is invalid. All Service Fabric enumerations have the invalid type. The value is zero. - Stateless - Does not use Service Fabric to make its state highly available or reliable. The value is 1. - Stateful - Uses Service Fabric to make its state or part of its state highly available and reliable. The value is 2. 

## Enum

* `INVALID` (value: `'Invalid'`)

* `STATELESS` (value: `'Stateless'`)

* `STATEFUL` (value: `'Stateful'`)
