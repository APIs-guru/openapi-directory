# HostType

Specifies the type of host for main entry point of a code package as specified in service manifest. Possible values are following.    - Invalid - Indicates the type of host is not known or invalid. The value is 0.   - ExeHost - Indicates the host is an executable. The value is 1.   - ContainerHost - Indicates the host is a container. The value is 2. 

## Enum

* `INVALID` (value: `'Invalid'`)

* `EXEHOST` (value: `'ExeHost'`)

* `CONTAINERHOST` (value: `'ContainerHost'`)
