# FabricReplicaStatus

Specifies the status of the replica. Possible values are following.   - Invalid - Indicates that the read or write operation access status is not valid. This value is not returned to the caller.   - Down - Indicates that the replica is down.   - Up - Indicates that the replica is up. 

## Enum

* `INVALID` (value: `'Invalid'`)

* `DOWN` (value: `'Down'`)

* `UP` (value: `'Up'`)
