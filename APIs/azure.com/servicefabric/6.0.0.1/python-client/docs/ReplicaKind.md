# ReplicaKind

The role of a replica of a stateful service. Possible values are following.   - Invalid - Represents an invalid replica kind. The value is zero.   - KeyValueStore - Represents a key value store replica. The value is 1 

## Enum

* `INVALID` (value: `'Invalid'`)

* `KEYVALUESTORE` (value: `'KeyValueStore'`)
