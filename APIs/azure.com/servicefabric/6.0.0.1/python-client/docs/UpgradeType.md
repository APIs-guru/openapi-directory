# UpgradeType

The type of upgrade out of the following possible values.      - Invalid - Indicates the upgrade kind is invalid. All Service Fabric enumerations have the invalid type. The value is zero.     - Rolling - The upgrade progresses one upgrade domain at a time. The value is 1.     - Rolling_ForceRestart - The upgrade gets restarted by force. The value is 2. 

## Enum

* `INVALID` (value: `'Invalid'`)

* `ROLLING` (value: `'Rolling'`)

* `ROLLING_FORCE_RESTART` (value: `'Rolling_ForceRestart'`)
