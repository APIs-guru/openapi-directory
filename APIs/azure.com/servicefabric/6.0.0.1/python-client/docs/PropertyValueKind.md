# PropertyValueKind

The kind of property, determined by the type of data. Following are the possible values.  - Invalid - Indicates the property is invalid. All Service Fabric enumerations have the invalid type. The value is zero. - Binary - The data inside the property is a binary blob. The value is 1. - Int64 - The data inside the property is an int64. The value is 2. - Double - The data inside the property is a double. The value is 3. - String - The data inside the property is a wstring. The value is 4. - Guid - The data inside the property is a guid. The value is 5. 

## Enum

* `INVALID` (value: `'Invalid'`)

* `BINARY` (value: `'Binary'`)

* `INT64` (value: `'Int64'`)

* `DOUBLE` (value: `'Double'`)

* `STRING` (value: `'String'`)

* `GUID` (value: `'Guid'`)
