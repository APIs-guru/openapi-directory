# UpgradeKind

The kind of upgrade out of the following possible values.      - Invalid - Indicates the upgrade kind is invalid. All Service Fabric enumerations have the invalid type. The value is zero.     - Rolling - The upgrade progresses one upgrade domain at a time. The value is 1 

## Enum

* `INVALID` (value: `'Invalid'`)

* `ROLLING` (value: `'Rolling'`)
