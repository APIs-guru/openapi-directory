## SDK Available Operations

### jobs

* `youtubereportingJobsCreate` - Creates a job and returns it.
* `youtubereportingJobsDelete` - Deletes a job.
* `youtubereportingJobsGet` - Gets a job.
* `youtubereportingJobsList` - Lists jobs.
